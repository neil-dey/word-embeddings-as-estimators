This repository contains the source code for the experiments in our paper "Word Embeddings as Statistical Estimators."

The directory structure is as follows:

- workflow.sh
	- Contains the code necessary to replicate all simulations and experiments from the paper, as well as technical information on runtimes and CPU specifications.

- Brown/
	- Contains the Brown Corpus and the code to replicate Figure 1.

- generative_model/
	- Contains the code necessary to replicate Section 5 of the paper.

- IMDB_Final/
	- Contains the code necessary to replicate Section 6 of the paper.
